# Senaite Sample Importer

**Bulk sample import tool for SENAITE and Bika LIMS**

This add-on allows laboratories to quickly import large numbers of samples from CSV files into SENAITE / Bika LIMS.

### Features

- Import multiple samples at once
- Supports CSV files
- Validation and error reporting
- Compatible with SENAITE core and Ingwe Bika LIMS

### Installation

Add `senaite.sampleimporter` to your SENAITE / Bika LIMS instance and restart.
